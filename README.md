## 安装

### Mac

下载Mac安装包，安装JumpServer.dmg并将JumpServer.app放入Applications中，双击打开JumpServer.app完成自定义协议。

### Win

下载Win安装包，双击JumpServer.exe 文件（需要等待10几秒钟）。

## 卸载

### Windows

控制面板直接卸载即可。

### Mac

移除JumpServer.app。
